Files Library
=============

`namespace LFiles`

`int getFileInformation()`

- `const char* fileName` - Path of file or directory
- `bool *exists`
- `bool *isFile`
- `bool *isDir`

Building
--------

1. Install [Retro68](https://github.com/autc04/Retro68);
2. `cd` into this directory;
3. `make clean`
4. `make`
5. `make install`
